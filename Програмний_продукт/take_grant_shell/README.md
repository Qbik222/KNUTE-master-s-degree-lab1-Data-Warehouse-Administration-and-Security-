# Операційна оболонка з моделлю Take-Grant

## Опис

Реалізація дискреційного управління доступом на основі моделі Take-Grant.

## Структура проекту

```
take_grant_shell/
├── main.py                 # Головний файл запуску
├── modules/
│   ├── __init__.py
│   ├── auth.py             # Реєстрація та авторизація
│   ├── objects.py          # Ідентифікація об'єктів
│   ├── access_graph.py     # Граф доступу (Take-Grant)
│   ├── security_kernel.py  # Ядро безпеки (перевірка доступу)
│   ├── operations.py       # Операції над об'єктами
│   ├── admin.py            # Модуль адміністратора
│   ├── audit.py            # Журнал аудиту
│   └── cli.py              # Консольний інтерфейс
├── data/
│   └── system.json         # Зберігання даних системи
└── logs/
    └── audit.log           # Журнал подій
```

## Встановлення

```bash
cd take_grant_shell
python3 main.py
```

## Використання

Після запуску програми ви побачите консольний інтерфейс з командами:
- `register` - реєстрація нового користувача
- `login` - авторизація
- `create` - створення об'єкта
- `take` - операція take
- `grant` - операція grant
- `access` - перевірка доступу
- `list` - список об'єктів
- `admin` - адміністративні команди
- `help` - довідка
- `exit` - вихід
